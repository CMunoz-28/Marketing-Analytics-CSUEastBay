# Mobile Marketing & AI – UI Concept Design

**Course:** Mobile Marketing & AI  
**Program:** Master of Science in Marketing Analytics  
**Institution:** California State University, East Bay  

---

## Project Overview

This project presents a **conceptual mobile UI redesign** created as part of a graduate-level Mobile Marketing & AI course. The objective was to select an existing brand and **pitch a redesigned user interface** that improves usability, engagement, and mobile-first marketing effectiveness.

The selected brand was **Duo Mobile**, used strictly as a **case example for academic analysis and design exploration**.

---

## Project Structure
- `design/figma/` – Editable Figma source file for the mobile app mockup  
- `reports/` – Final PDF presentation and documentation  

---

## Project Objectives

- Evaluate an existing mobile app experience from a UX and marketing perspective  
- Design a **conceptual UI mockup** aligned with mobile engagement best practices  
- Incorporate personalization and AI-informed design considerations  
- Demonstrate how UI decisions can support mobile marketing and user adoption  

---

- **Figma Design File**
